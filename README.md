# Finance Tracker

This Finance Tracker will be a way to track spending. This app is helpful for anyone looking to minimize costs and see how much they are actually spending money throughout a time period. 

## Features 

- User Functions: Registration, Log in, Update profile, and Delete account.
- Transations: Create transation, View transations, Edit transations, and Delete transations.
- Search Functions: Users are able to search all transations within a certain time period.
